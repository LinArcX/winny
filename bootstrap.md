## PowerShell
- Create new alias:
`new-alias grep findstr`

- FullScree current window:
`mode 300`

- Add new variable to environment variables list:
$env:Path += ";C:\tools\neovim\Neovim\bin" 

- Refresh environment variables without close terminal:
`refreshenv`
